# goserv

Simple local go server into input path.

## Installation

-   Go [https://golang.org](https://golang.org)
-   `go install github.com/gouvinb/goserv`

## Usage

### Usage of goserv

Into terminal :

```bash
goserv #[FLAG arg...]
```

### List of task
-   `path`

    -   which directory to create doc server (default "./")

-   `port`

    -   port of server (default "4000")
